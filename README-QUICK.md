# AgroBot

AI-Powered Smart Farming Assistant providing intelligent recommendations for crop selection, fertilizer management, irrigation, pest control, and sustainable farming practices.

## Quick Start

### Setup
1. Navigate to the project directory
2. Create Python virtual environment: `python -m venv .venv`
3. Activate it (Windows): `.\.venv\Scripts\activate`
4. Install dependencies: `cd backend && pip install -r requirements.txt`
5. Run backend: `python run.py`
6. Open `frontend/index.html` in your browser

### Features
- AI-powered chat for farming advice
- Crop recommendations
- Fertilizer and irrigation guidance
- Pest management assistance
- Soil health analysis
- Voice input/output support
- Multilingual support (English, Hindi, Telugu)
- Professional farmer dashboard

### Access
- Frontend: `http://localhost:8000` (or open frontend/index.html directly)
- Backend API: `http://localhost:10000`
