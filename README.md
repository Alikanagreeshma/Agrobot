# 🌾 AgroBot

AgroBot is an AI-powered multilingual agriculture assistant designed to support Indian farmers with crop recommendations, soil management, irrigation guidance, fertilizer suggestions, pest control, and sustainable farming practices.

## 🚀 Features

* 🤖 AI-powered farming assistance using OpenRouter
* 🌱 Crop recommendation support
* 💧 Irrigation guidance
* 🧪 Soil and fertilizer recommendations
* 🐛 Pest and disease management advice
* 🎤 Voice input support
* 🔊 Voice output support
* 🌍 Multilingual support (English, Hindi, Telugu)
* 📷 Crop and soil image upload interface
* 📱 Farmer-friendly user interface

## 🛠️ Tech Stack

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Alikanagreeshma/Agrobot.git
cd Agrobot
```

### Install Dependencies

```bash
pip install -r backend/requirements.txt
```

### Configure Environment Variable

```bash
OPENROUTER_API_KEY=your_api_key_here
```

### Run Backend

```bash
cd backend
python app.py
```

### Open Frontend

Open `index.html` in your browser.
